# Elixir Education Ecosystem

A comprehensive, conversion-focused education platform featuring split-screen navigation between an Institute (JEE/NEET coaching) and an International School.

## Features Implemented

### Phase 1 (MVP) - Complete
- **Multi-step Inquiry Form** with validation and progress indicators
- **FAQ Accordion** sections for both Institute and School
- **Testimonials** with rotating parent quotes and star ratings
- **Fee Structure** with expandable payment options and scholarship info
- **Fee Calculator** with interactive sliders

### Lead Generation
- Floating WhatsApp integration
- Live chat widget with quick replies
- Newsletter subscription forms
- Callback request system
- Sticky CTA bar on scroll

### Content Sections
- **Institute Page:**
  - Results Gallery with filterable toppers
  - Faculty Carousel with credentials
  - Course cards with features
  - Live Batch Schedule with countdowns
  
- **School Page:**
  - Virtual Campus Tour modal
  - Facilities Gallery with lightbox
  - Academic Programs comparison
  - Admission Process timeline
  - Events Calendar
  - Location Map with directions

### UI/UX Enhancements
- Cookie consent banner (GDPR-friendly)
- Toast notifications for form submissions
- Scroll progress indicator
- Breadcrumb navigation
- Animated counters for statistics
- Smooth scroll-triggered animations
- Ripple effect on buttons

### Accessibility
- Skip links for keyboard navigation
- ARIA labels and roles
- Focus visible states
- Reduced motion support
- High contrast mode support
- Print stylesheet

### Performance
- Lazy loading ready
- Image preloading
- CSS animations with GPU acceleration
- Efficient scroll handlers with requestAnimationFrame

## Tech Stack
- Pure HTML5, CSS3, JavaScript (ES6+)
- No frameworks or dependencies (except Font Awesome icons)
- CSS Grid & Flexbox layouts
- CSS Custom Properties (variables)
- LocalStorage for bookmarks and preferences
- Intersection Observer API for animations

## Browser Support
- Chrome, Firefox, Safari, Edge (latest versions)
- Mobile responsive (tested on iOS/Android)

## Getting Started
Simply open `index.html` in a browser. No build process required.
